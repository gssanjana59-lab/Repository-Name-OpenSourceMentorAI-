# 🚀 OpenSource Mentor AI

OpenSource Mentor AI helps developers analyze their GitHub profiles and receive personalized recommendations to improve their open-source journey.

## Features

* GitHub Profile Analysis
* Repository Statistics
* Total Stars Counter
* Total Forks Counter
* Programming Language Analysis
* Open Source Mentor Score
* Personalized Recommendations

## Tech Stack

### Frontend

* React
* Vite

### Backend

* Node.js
* Express.js

### API

* GitHub REST API

## Installation

### Backend

```bash
cd server
npm install
node app.js
```

### Frontend

```bash
cd client
npm install
npm run dev
```

## Usage

1. Enter a GitHub username.
2. Click Analyze.
3. View profile insights, stars, forks, languages, mentor score, and recommendations.
